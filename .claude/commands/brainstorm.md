# /brainstorm - Ideation Mode

Generate creative ideas and explore possibilities.

## Instructions

1. **Understand the problem** - What are we trying to solve?
2. **Generate options** - Come up with multiple approaches
3. **Evaluate tradeoffs** - Pros/cons of each
4. **Recommend** - Suggest best option with reasoning

## Output Format

```markdown
## Brainstorm: [Topic]

### Problem Statement
[Clear description of what we're solving]

### Constraints
- [constraint 1]
- [constraint 2]

### Option 1: [Name]
**Description:** [How it works]
**Pros:**
- [advantage]
**Cons:**
- [disadvantage]
**Effort:** [Low/Medium/High]

### Option 2: [Name]
...

### Option 3: [Name]
...

### Comparison Matrix
| Criteria | Option 1 | Option 2 | Option 3 |
|----------|----------|----------|----------|
| Speed    | ⭐⭐⭐   | ⭐⭐     | ⭐       |
| Cost     | ⭐       | ⭐⭐     | ⭐⭐⭐   |
| Quality  | ⭐⭐     | ⭐⭐⭐   | ⭐⭐     |

### Recommendation
**Go with Option X because...**
[Reasoning]

### Open Questions
- [Question that needs more info]
```

## Rules
- Generate at least 3 options
- Include unconventional ideas
- Be specific, not generic
- Consider MVP vs ideal solution
